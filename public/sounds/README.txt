Банк звуков «Релакс-окна» (ФОРТОЧКА)
=====================================
Положите сюда лицензионно-чистые аудиофайлы (CC0 / public domain).
Источники: freesound.org (фильтр CC0), pixabay.com/sound-effects, mixkit.co

Поддерживаемые имена (любое из расширений .mp3 / .ogg / .wav):
  meow1.mp3, meow2.mp3, meow3.mp3   — мяуканье (можно несколько, выбирается случайно)
  purr.mp3                          — мурчание (зациклится)
  squeak.mp3                        — писк мыши
  wings.mp3                         — взмах крыльев птицы
  bee.mp3                           — жужжание пчелы (зациклится)
  bumble.mp3                        — жужжание шмеля (зациклится)
  wind.mp3                          — ветер (зациклится)
  birds.mp3                         — пение птиц фоном (зациклится)

Любого файла может не быть — тогда для него играет процедурный синтез.
Файлы подхватываются автоматически при открытии Релакс-окна.

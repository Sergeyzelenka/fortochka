Спрайты котов для «Релакс-окна» (ФОРТОЧКА)
============================================
Положите сюда PNG-кадры котов с ПРОЗРАЧНЫМ фоном. Кот смотрит ВПРАВО
(движок сам отзеркалит для движения влево).

Вариант А — отдельные кадры (просто):
  cat1_walk_0.png, cat1_walk_1.png, ... cat1_walk_7.png   (цикл ходьбы/бега)
  cat1_sit_0.png                                          (сидит)
  cat1_wash_0.png, cat1_wash_1.png                        (умывается)
  cat1_sleep_0.png                                        (спит)
  cat1_roll_0.png                                         (валяется)
Несколько котов: cat2_*, cat3_* — у каждого свой набор и окрас.

Вариант Б — спрайт-лист (одна картинка-полоса):
  cat1_walk.png + рядом cat1_walk.json с {frames, frameWidth, frameHeight}

Размер кадра: ~128–256px по высоте, кот занимает ~80% кадра.
Где взять бесплатные (CC0): itch.io (free game assets), opengameart.org, kenney.nl

Если папка пуста — движок рисует прежних «кодовых» котов (фолбэк).
